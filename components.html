Bootstrap Components & Dynamic Features

Documentation of all Bootstrap components and JavaScript features implemented in Patrick Meehan's portfolio website

Bootstrap Components Used

Layout & Grid System
  • Container - Responsive container for content
  • Row & Columns - Bootstrap grid system (col-lg-8, col-md-7, etc.)
  • Responsive Utilities - justify-content-center, text-center, etc.
Navigation Components
  • Navbar - Fixed-top responsive navigation
  • Navbar Brand - Logo/brand area
  • Navbar Toggler - Mobile hamburger menu
  • Navbar Collapse - Collapsible navigation menu
  • Nav Links - Navigation menu items
Content Components
  • Cards - Multiple card components for content sections
  • Card Body - Card content areas
  • Badges - Technology skill tags
  • Buttons - Primary, outline, and success button variants
  • Images - Responsive images with rounded class
Interactive Components
  • Modal - Contact information modal dialog
  • Modal Header - Modal title area
  • Modal Body - Modal content area
  • Button Close - Modal close button

Dynamic JavaScript Features

Theme & Visual Effects
  • Theme Switcher - Cycles through 4 color schemes (nature, ocean, sunset, forest)
  • Dark/Light Mode Toggle - Switches between dark and light themes
  • Scroll Progress Bar - Shows page scroll progress at top
  • Back to Top Button - Appears after scrolling 300px
  • Typing Animation - Name types out character by character
Interactive Elements
  • Dog Image Carousel - Cycles through personal dog photos
  • Smooth Scrolling - Back to top with smooth behavior
  • Responsive Design - Mobile-first responsive layout
CSS Enhancements
  • CSS Custom Properties - Dynamic color scheme variables
  • Hover Effects - Card lift and badge scale animations
  • Transitions - Smooth animations throughout
  • Custom Styling - Nature-inspired design theme
Technical Implementation
  • Event Listeners - DOM content loaded, scroll, click events
  • State Management - Theme index tracking, dark mode state
  • Dynamic Calculations - Scroll percentage, viewport measurements
  • Element Manipulation - Style updates, content changes
